# Space Invaders

A Micro:bit arcade game inspired by Space Invaders.

## What it does

- Moves a player sprite left and right using buttons
- Shoots at an enemy sprite
- Uses score and lives
- Shows basic collision detection

## Controls

- **Button A:** Move left
- **Button B:** Move right
- **Buttons A+B:** Shoot

## Coding Concepts

- LED sprites
- Button events
- Loops
- Collision detection
- Score and lives

## What I learned

I learned how a game can be built using sprites, movement, shooting, and simple rules.
