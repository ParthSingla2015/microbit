# Wimpy Kid 20

A book-inspired Micro:bit project based on the fun style of the Wimpy Kid stories.

## What it does

- Shows messages when buttons are pressed
- Displays LED face patterns
- Reacts to shake gestures
- Connects reading and coding in a playful way

## Controls

- **Button A:** Show one message and face
- **Button B:** Show another message and face
- **Shake:** Show a surprised icon and face

## Coding Concepts

- Button events
- Gesture input
- LED icons and patterns
- Text display
- Book-inspired creative coding

## What I learned

I learned that coding can be connected with books and storytelling, not just games.
