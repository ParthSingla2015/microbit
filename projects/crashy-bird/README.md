# Crashy Bird

A Micro:bit obstacle game inspired by flying-through-gaps arcade games.

## What it does

- Controls a bird sprite on the LED grid
- Creates moving obstacles
- The player must avoid crashing into obstacles
- The game ends when the bird touches an obstacle

## Controls

- **Button A:** Move bird up
- **Button B:** Move bird down

## Coding Concepts

- LED sprites
- Arrays of obstacles
- Loops
- Random gap generation
- Collision detection
- Game-over logic

## What I learned

I learned how obstacle games work by creating moving objects, gaps, and collision rules.
