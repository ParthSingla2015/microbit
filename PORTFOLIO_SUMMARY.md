# Portfolio Summary

This repository presents **9 Micro:bit projects by Parth Singla** in one clean, organised portfolio.

## Portfolio Purpose

The purpose of this repository is to show Parth's hands-on learning in:

- Micro:bit coding
- Game design
- Randomness and probability
- Sensors and gestures
- Music and physical computing
- Book-inspired creative coding

## Project List

| Project | Key Idea |
|---|---|
| Space Invaders | Arcade game using sprites, score, lives, and collision detection |
| Memory Game | Sequence memory game with levels and answer checking |
| Coin Flipper | Probability project using random outcomes |
| Rock Paper Scissors | Gesture-based random choice game |
| Dot Chaser | Gesture movement on a 5x5 LED grid |
| Snap the Dot | Timing and reflex game |
| Crashy Bird | Obstacle game using arrays and collision logic |
| Compass and Music Maker | Compass, touch pins, gestures, and music |
| Wimpy Kid 20 | Book-inspired creative coding project |

## Review Highlights

The strongest projects to review first are:

1. **Space Invaders** — strongest arcade-style game
