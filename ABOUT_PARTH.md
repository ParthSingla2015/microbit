# About Parth Singla

Hi, I’m **Parth Singla**, a 10-year-old eco-storyteller, builder, Micro:bit learner, book reviewer, and changemaker from India. I love to read, invent, build, code, and inspire other children through stories, STEM projects, LEGO-style builds, environmental ideas, and short educational content.

My work focuses on clean air, climate action, ocean protection, reading, kindness, and child-friendly storytelling. I try to make learning fun by connecting books, coding, creativity, and action.

**Motto:** Read. Build. Code. Imagine. Act.

## Micro:bit and Coding Work

This repository presents my Micro:bit coding work in one structured portfolio. These projects show how a small device like the BBC Micro:bit can be used to learn game design, logic, sensors, movement, music, and creative expression.

| Project | What it shows |
|---|---|
| [Space Invaders](./projects/space-invaders) | Arcade-style game design using sprites, score, lives, shooting, and collision detection |
| [Memory Game](./projects/memory-game) | Logic, sequence generation, answer checking, levels, and increasing difficulty |
| [Coin Flipper](./projects/coin-flipper) | Randomness and probability through a digital coin toss |
| [Rock Paper Scissors](./projects/rock-paper-scissors) | Gesture input and random choice selection |
| [Dot Chaser](./projects/dot-chaser) | Movement on a 5x5 LED grid using gestures and coordinates |
| [Snap the Dot](./projects/snap-the-dot) | Timing, reaction, score tracking, and game-over logic |
| [Crashy Bird](./projects/crashy-bird) | Obstacle generation, arrays, movement, and collision detection |
| [Compass and Music Maker](./projects/compass-and-music-maker) | Compass heading, gestures, touch pins, direction display, and music playback |
| [Wimpy Kid 20](./projects/wimpy-kid-20) | Book-inspired creative coding using messages, icons, and gestures |

## What I Do

- Build LEGO-style machines and motorized Technic models

- Create child-friendly awareness content on reading, climate action, kindness, and sustainability
- Share learning through public profiles and project portfolios

## Learning Style

I learn best by making:

1. Try an idea
2. Build a small version
3. Test it
4. Improve it
5. Explain what was learned
6. Share it so other children feel inspired
