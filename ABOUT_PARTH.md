# About Parth Singla

**Parth Singla** is a young student, STEM builder, Micro:bit learner, book reviewer, and creative storyteller who enjoys turning ideas into hands-on projects.

His work brings together **coding, reading, building, imagination, and problem-solving**. Through his projects, Parth explores how technology can make learning more fun, interactive, and meaningful for children.

## What Parth Creates

Parth enjoys working on different kinds of creative learning projects, including:

- Micro:bit games
- STEM experiments
- LEGO and mechanical builds
- Book-inspired creative projects
- Coding activities for learning and fun
- Environmental and sustainability-themed ideas
- Book reviews and storytelling projects

## Micro:bit and Coding Work

This repository shows Parth's Micro:bit coding work. His projects include games, logic challenges, sensor-based activities, music experiments, and book-inspired coding ideas.

Some of his projects include:

- **Space Invaders** — an arcade-style Micro:bit game using sprites, score, lives, and collision detection
- **Memory Game** — a logic game based on remembering and repeating sequences
- **Coin Flipper** — a simple probability project using random outcomes
- **Rock Paper Scissors** — a classic game recreated using Micro:bit gestures and randomness
- **Dot Chaser** — a movement-based game using the Micro:bit LED grid
- **Snap the Dot** — a reflex game based on timing
- **Crashy Bird** — an obstacle-based arcade game
- **Compass and Music Maker** — a project combining compass direction, gestures, touch pins, and music
- **Wimpy Kid 20** — a book-inspired creative coding project

These projects show Parth's interest in learning by doing. They also show how simple hardware like the BBC Micro:bit can be used to understand coding, game design, sensors, maths, and creativity.
