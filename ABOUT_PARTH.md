# About Parth Singla

**Parth Singla** is a young STEM builder, Micro:bit learner, book reviewer, eco-storyteller, and creative changemaker from India. His work brings together **coding, reading, LEGO-style building, environmental awareness, imagination, and hands-on problem-solving**.

Parth uses creativity as a bridge between books and technology. He reads stories, reviews books, builds STEM projects, codes Micro:bit games, experiments with sensors, and shares ideas that encourage other children to read, build, code, imagine, and act.

## Micro:bit and Coding Work

This repository presents Parth's Micro:bit coding work in one structured portfolio. The projects show how he uses a small device like the BBC Micro:bit to understand game design, logic, sensors, movement, music, and creative expression.

### Projects in this repository

| Project | What it shows |
|---|---|
| [Space Invaders](./projects/space-invaders) | Arcade-style game design using sprites, score, lives, shooting, and collision detection |
| [Memory Game](./projects/memory-game) | Logic, sequence generation, answer checking, levels, and increasing difficulty |
| [Coin Flipper](./projects/coin-flipper) | Randomness and probability through a digital coin toss |
| [Rock Paper Scissors](./projects/rock-paper-scissors) | Gesture input and random choice selection |
| [Dot Chaser](./projects/dot-chaser) | Movement on a 5x5 LED grid using gestures and coordinates |
| [Snap the Dot](./projects/snap-the-dot) | Timing, reaction, score tracking, and game-over logic |
| [Crashy Bird](./projects/crashy-bird) | Obstacle generation, arrays, movement, and collision detection |
| [Compass and Music Maker](./projects/compass-and-music-maker) | Compass heading, gestures, touch pins, direction display, and music playback |
| [Wimpy Kid 20](./projects/wimpy-kid-20) | Book-inspired creative coding using messages, icons, and gestures |

### Coding concepts Parth is learning

- Event-based programming
- Button and gesture inputs
- LED sprite movement
- Random number generation
- If/else logic and conditions

## Books, STEM, and Building

Parth's creative identity is built around the idea that books can lead to building. His Bookosmia article **Adventures, Magic & Me: Journey Through Books** explains how reading inspired him to create hands-on projects, including a DIY mechanical music machine and a motorised LEGO Technic Peugeot 9X8 car.

This connection is important because it shows a learning pattern:

1. Read a story or idea
2. Imagine how it could work in the real world
3. Build or code a small version
4. Share the learning with others

## Learning Style

Parth learns best by making. His work reflects a practical maker mindset:

- Try an idea
- Build a small version
- Test it
- Improve it
- Explain what was learned
- Share it so other children feel inspired

This approach helps him understand coding and STEM concepts in a practical and joyful way.

## Why This Repository Matters

This Micro:bit repository is one part of Parth's larger creative journey. It shows the coding side of his work, while his writing and public profiles show the storytelling, reading, environmental, and communication side.

Together, these activities show Parth as a young learner who is exploring how children can use simple tools, stories, and imagination to create meaningful projects.
